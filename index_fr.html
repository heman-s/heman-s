Description:

In this Java Program, a simulation of six different species is represented, showing how they co-exist within an ecosystem, with some being prey and others being predators. Each species have corresponding activities for during the day and the night, and can be affected by age, hunger, disease, weather and overcrowding. Each species is able to move and breed when certain conditions permit, may sleep at different times, may become infected or cured accordingly, may die and provide a different food value for its predators. These figures are randomly generated, however, include a bias to more accurately represent real life conditions.

Auteurs Originals: David J. Barnes, Michael Kölling
Contributeurs: Heman Seegolam, Bhavik Gilbert Dans cette application de Java, on représente une simulation de six espèces differentes que montre comment ils coexistent dans un écosystème, où certains espèces sont des prédateurs et les autres sont des proies. Chaque espèce est capable de se déplacer et se reproduire quand les conditions les permettent.

Description:

The property viewer application displays 4 distinct panels which each give insight into the London Property Marketplace. The user can select a preferred price range using a rangeslider (or use the default range already set), which allows them to view properties on map that displays a boroughs corresponding, available properties. There is a statistics panel that provides statistical information on properties that fall within the chosen price range, and also a favourites panel that displays the properties which have been favourited by the user. The GUI has various, additional features such as a menubar, an overall price calculator, a property selection scroll pane, the ability to display the location of each property on OpenStreetMaps, a search bar for distinct values of each property, as well as a darkmode-lightmode implementation and a Weather API.

Auteurs Originals: David J. Barnes, Michael Kölling
Contributeurs: Heman Seegolam, Bhavik Gilbert, Nina Peire, Lavish Kumar

Description:

In this Python application, users can log in and sign up as a teacher, parent or student. Logins are checked against a hashed database to verify credentials and sign ups must be verified by teachers. Teachers are able to create quizzes - featuring various question types and other options, play quizzes, view classes and students and monitor their progress. Students are able to play quizzes created by the teacher or play quizzes that have been pre-created (where questions will be randomised and solved by using a corresponding equation for the question). Parents are able to monitor the progress of their children.

Auteurs Originals: Heman Seegolam

Description:

In this Java text-based, role-play game, the user is trapped in a nightmare, and is required to move between several locations, use and carry different items, interact with non-player characters, to solve clues and ultimately escape. The user is faced with riddles and must find different keys and codes to unlock various items which shall help them escape. Different items perform different tasks and each location incorporates different scenarios and clues that may have to be used across rooms.
