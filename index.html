    Personal Project

    Description:

    This Java program simulates six distinct species coexisting within an ecosystem, with some acting as predators and others as prey. Each species exhibits specific behaviors during the day and night, and their survival is influenced by factors such as age, hunger, disease, weather, and overcrowding. The species can move, breed under certain conditions, sleep at different intervals, become infected or cured, and eventually die, offering varying food values to their predators. Although the simulation involves random generation, it incorporates biases to more realistically reflect natural conditions.

    Original Authors: David J. Barnes, Michael Kölling

    Description:

    The property viewer application features four distinct panels, each offering insights into the London property market. Users can select a preferred price range using a range slider (or default settings) to filter and view properties on a map that highlights boroughs and available listings. A statistics panel provides detailed information on properties within the selected price range, while a favorites panel displays properties the user has marked as favorites. The GUI includes several additional features, such as a menubar, an overall price calculator, a property selection scroll pane, and the ability to display property locations using OpenStreetMap. It also offers a search bar for filtering properties by specific criteria, a dark mode/light mode toggle, and integration with a weather API.

    Original Authors: David J. Barnes, Michael Kölling
    Contributors: Heman Seegolam, Bhavik GilbertGo to website , Nina Peire, Lavish Kumar

    Description:

    In this Python application, users can sign up and log in as a teacher, parent, or student. Login credentials are securely verified against a hashed SQL database, and new sign-ups require teacher approval. Teachers have the ability to create quizzes with various question types and options, assign quizzes, manage classes and students, and track student progress. Students can participate in quizzes created by their teachers or attempt pre-generated quizzes, where questions are randomized and solved using corresponding equations. Parents can monitor their children's progress, providing an overview of their academic performance.

    Original Authors: Heman Seegolam

    Description:

    In this Java text-based role-playing game, the user finds themselves trapped in a nightmare and must navigate through various locations, collect and use different items, and interact with non-player characters to solve puzzles and ultimately escape. The game challenges the player with riddles and requires them to discover keys and codes to unlock essential items for their escape. Each item serves a unique purpose, and the locations present distinct scenarios and clues, some of which may need to be applied across multiple rooms to progress in the game.
